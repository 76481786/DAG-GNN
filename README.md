# DAG-GNN

Code for DAG-GNN work

## Getting Started

### Prerequisites

```
PyTorch >1.0
```


## How to Run 

Synthetic data experiments 

### Synthetic Experiments

CHOICE = linear, nonlinear_1 or nonlinear_2 

```
python train.py --graph_linear_type=<CHOICE>
```
